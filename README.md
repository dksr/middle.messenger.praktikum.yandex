### Команды для запуска
Билд проекта + запуск сервера localhost:3000, раздача статики из dist:
> npm run start 
> 
Запуск parcel в дев режиме на localhost:1234
>npm run dev
>
Билд проекта в папку dist
>npm run build
> 
ESLint
>npm run eslint
> 
Stylelint
>npm run stylelint
>
